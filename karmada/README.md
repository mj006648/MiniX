# Karmada 실험 기록 이동 안내

Karmada 기반 ScaleX-POD 멀티클러스터 실험 기록과 매니페스트는 운영 런북 저장소로 이동했다.

- 이동 위치: [netai-devsecops-runbook/kubernetes/karmada](https://github.com/mj006648/netai-devsecops-runbook/tree/main/kubernetes/karmada)
- 실험 진행표: [Karmada 실험 진행표](https://github.com/mj006648/netai-devsecops-runbook/blob/main/kubernetes/karmada/experiments/README.md)
- 운영 절차: [Karmada 운영 Runbook](https://github.com/mj006648/netai-devsecops-runbook/tree/main/kubernetes/karmada/runbooks)

MiniX 저장소에는 Karmada 실험의 출발점과 연결 링크만 남긴다.
이후 Karmada 관련 실험, 장애 대응 기록, 운영 절차는 `netai-devsecops-runbook`에서 관리한다.
